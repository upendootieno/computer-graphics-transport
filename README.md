## Toka Kwa Block

## TRAFFIC SIMULATION PROJECT 1 *

Traffic simulation project 1 (informally Toka Kwa Block)* is an experimental learning project for simulating different traffic environments and conditions.


### System Requirements

The project must be able to run on a computer with the following specifications or better;

- GPU: OpenGL > 3.2
- Processor: Intel Core i3 processor / AMD ryzen 3 or better

### Primary Stack

The project makes use of the following tools;

- Computation: Matlab R2025b
- Simulation: Unity 6.2
- Modeling: Blender > 2.93
